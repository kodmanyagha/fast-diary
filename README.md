# Fast Diary

Diary app built with Druid library and Rust language.

```bash
# Linux Bash
RUST_LOG='debug' cargo watch -x run
```

```powershell
# Windows powershell
$env:RUST_LOG='debug'; cargo watch -x run; $env:RUST_LOG=$null
```
